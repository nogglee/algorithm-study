# Algorithm Study Log

수학적 사고력을 기르기 위해 스터디를 수행합니다.

---

## 💡 Study Principles

- 요구사항을 파악하여 **나만의 언어**로 정리하여 문제 이해력을 높입니다.
- 바로 정답을 찾기보다 **직접 시도하고 막힌 지점을 기록**합니다.
- 풀이가 끝나면 **문제의 핵심은 무엇이었지** 짧게 정리합니다.

---

## 🏁 Execution Criteria

1. 문제를 읽고 요구사항/입출력을 분석한다.
2. 접근 전략을 먼저 글로 정리한다
3. 코드로 구현하며 검증한다
4. 헷갈린 포인트/실수/대안을 기록한다
5. 배운 점을 한 줄로 남긴다

---
