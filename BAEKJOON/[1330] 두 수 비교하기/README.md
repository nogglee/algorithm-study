# BOJ [1330] 두 수 비교하기
<sup>[문제 바로가기](https://www.acmicpc.net/problem/1330)</sup>

## 문제 분석
1. A와 B를 공백 한칸으로 구분하여 입력 받는다.
2. 두 변수의 크기를 비교하여 기호를 출력한다.

---

## 접근 방법
1. 두 수를 입력받아 공백을 기준으로 분리한다.
2. 두 수의 값을 비교하는 조건을 걸어 결과값을 출력한다.

---

## 배운 점
출력시간이 짧은(28ms) 분의 풀이법을 발견하여 분석해보았다.
```python
A, B = map(int, input().split())
print("==" if A == B else "><"[A < B])
```

해석:
1. A와 B가 같으면 '=='를 출력한다.
2. A와 B가 같지 않으면 문자열을 인덱싱하여 결과값을 출력한다.
    1. 이때, 조건은 A는 B보다 작다.
    2. 조건의 결과가 True라면 인덱스 1, False라면 인덱스 0을 출력한다.
        1. A < B is True => 1 (<)
        2. A < B is False => 0 (>)

## 결과
- 시간: 36ms
- 메모리: 32412KB
