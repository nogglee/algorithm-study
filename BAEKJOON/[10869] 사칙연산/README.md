# 플랫폼명 [문제번호] 문제명
<sup>[문제 바로가기](https://www.acmicpc.net/problem/10869)</sup>

## 문제 분석
1. A와 B를 입력 받는다.
2. 첫째 줄에 두 변수의 합을 출력한다.
3. 둘째 줄에 두 변수의 차를 출력한다.
4. 셋째 줄에 두 변수의 곱을 출력한다.
5. 넷째 줄에 두 변수의 몫을 출력한다.
6. 다섯째 줄에 두 변수의 나머지를 출력한다.

---

## 접근 방법
1. 1000 문제에서 사용했던 값 입력 방식 사용
2. 한줄씩 순차적으로 값을 출력한다.

---

## 배운 점
- `A / B` : A 나누기 B의 실수 형태의 몫을 결과값으로 출력
- `A // B` : A 나누기 B의 정수 형태의 몫을 결과값으로 출력
- `A % B` : A 나누기 B의 나머지를 결과값으로 출력
